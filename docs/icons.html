Ikoner

Sådan bruger du Design System Icons

Med SVG sprite (anbefalet)

Du kan gemme alle ikoner som én fil og blot referere til ikonerne.

Download og gem ds-icon-sprites.svg et passende sted (PATH).

Herefter kan du blot indsætte SVG-tags og angive en reference til det ønskede ikon med use. F.eks.:

    
<svg><use href="PATH/icons.svg#arrow-right"></svg>

Med embedded SVG

Du kan indsætte ikoner direkte som SVG-kode. Det er en fordel, hvis du kun skal bruge enkelte ikoner.

Under hvert ikon kan du se SVG-koden, som du blot kan copy/paste ind i din HTML.

Ændre størrelse og farve på ikoner

Du kan ændre størrelse og farver på ikoner med CSS.
Størrelse ændres ved at justere height
Farve ændres ved at justere CSS variablen --ds-icon-color
